# java-calculator-precourse

## 기능 목록

- [ ] 사용자 입력 받기
    - [ ] 유효성 검사(빈 문자열)
- [ ] 구분자 유형별 처리 핸들링
    - [x] 기본 구분자
        - [x] 숫자 추출
    - [ ] 커스텀 구분자
        - [ ] 숫자 추출
- [ ] 숫자 덧셈
- [ ] 예외 처리

## 예외 케이스

### 공통

    - 빈 문자열 입력
    - 음수 입력
    - 구분자가 연속으로 나오는 경우(ex: ,, , ::)

### 기본 구분자

    - 기본 구분자 외의 문자 입력

### 커스텀 구분자

    - '//'로 시작 하지 않음
    - '\n'로 커스텀 구분자 지정을 마무리 하지 않음
    - 기본, 커스텀 구분자 외의 문자 입력
