# java-calculator-precourse

## 기능 목록

1. 사용자 입력(문자열) 받기
    - 유효성 검사
2. 구분자로 문자열 처리
    - 기본 구분자
    - 커스텀 구분자
3. 숫자 추출
4. 숫자 덧셈
5. 예외 처리

## 예외 케이스

### 공통

    - 빈 문자열 입력
    - 음수 입력
    - 구분자가 연속으로 나오는 경우(ex: ,, , ::)

### 기본 구분자

    - 기본 구분자 외의 문자 입력

### 커스텀 구분자

    - '//'로 시작 하지 않음
    - '\n'로 커스텀 구분자 지정을 마무리 하지 않음
    - 기본, 커스텀 구분자 외의 문자 입력
